## What is in this folder?

This folder contains coding standards, performance, and security rules, organized by language or framework applicability.

## When should the AI read files from here?

Read files from this folder **before** writing or modifying code. Always check applicable rules for the target language or framework.

## How to navigate and use the contents?

- **Structure**: **Flat** -> Single `list_dir` → global visibility.
- **Format**: Files named `common-[category].md` (applies to ALL code) or `[language]-[category].md` (only for specific language files).
- List the directory and read the rules relevant to the language and category of your current coding task.
