# 🚀 Quick Start

## For Human Operators
1. Set strategic goals in `BOARD.yaml`. Ensure `scopes`, `mission_ref` and `current_phase` are populated.
2. Switch to appropriate mode (STRICT/COLLAB/AUTO).
3. Let agents execute autonomously.

## For AI Agents
1. **Read `.brain/`** — Understand your persona, engine protocols, and 10-step flow.
2. **Check `BOARD.yaml`** — Detect current mode, active goals, and scopes.
3. **Scan `.scope/` Context** — Read context files matching your assigned goal.
4. **Follow Engine Protocols** — Use Navigator, Cataloger, and Router to find skills and execute tasks deterministically.
