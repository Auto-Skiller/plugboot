## What is in this folder?

This folder contains reusable scaffolding templates for creating new files, components, or project structures, partitioned by domain.

## When should the AI read files from here?

Read files from this folder when scaffolding new files, components, documents, or project structures to ensure consistency with workspace standards.

## How to navigate and use the contents?

- **Structure**: **Domain** -> Semantic routing by output type → prune search space.
- **Format**: `domain/` directories containing `[output-type].md` files (with placeholders).
- Route semantically by domain and the clear output expectation to find the correct template.
