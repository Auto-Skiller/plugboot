# Determine Next Steps

This skill allows the agent to determine or adjust the immediate next steps before taking action.

## Steps

1. Review the current state and the ultimate goal.
2. Identify dependencies and prerequisites.
3. Outline the immediate actions required.
4. Output a structured action plan.
