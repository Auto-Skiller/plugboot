# Generate Ideas

This skill allows the agent to generate ideas and explore multiple approaches when a path isn't clear.

## Steps

1. Define the problem statement.
2. Identify constraints and requirements.
3. Brainstorm potential solutions or approaches.
4. Output a categorized list of ideas.
