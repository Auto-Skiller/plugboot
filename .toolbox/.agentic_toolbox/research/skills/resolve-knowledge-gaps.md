# Resolve Knowledge Gaps

This skill allows the agent to resolve knowledge gaps, find external information, or read relevant documentation.

## Steps

1. Identify the topic and specific questions to answer.
2. Conduct research using available tools and resources.
3. Synthesize the findings.
4. Output the gathered information and answers.
