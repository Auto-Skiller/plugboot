# Write Docs

This skill allows the agent to produce written knowledge artifacts such as READMEs, guides, API specs, and tutorials.

## Steps

1. Analyze the source material and target audience.
2. Determine the appropriate format for the document.
3. Draft the content.
4. Review and finalize the written document.
