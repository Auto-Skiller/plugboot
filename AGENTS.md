# AGENTS

> [!IMPORTANT]
> This is the root pointer file for all agents landing in this workspace.
> The full operating system context lives inside `.brain/`.
>
> **The Three Pillars**:
> - `.brain/` — Identity, rules, persona, modes, orchestration logic
> - `.toolbox/` — Capability stack (agentic, business, engineering, life, studio)
> - `BOARD.yaml` — Unified state board (mode, goals, comms, scratchpad)
>
> **Toolbox Registry** (Index for navigating `.toolbox/`):
> - `.brain/.toolbox.control/.toolbox.registry/core_toolbox.registry`
> - `.brain/.toolbox.control/.toolbox.registry/extended_toolbox.registry`
>
> **Execution Pipelines**:
> - `_pipelines/hustler/` — Revenue discovery & execution
> - `_pipelines/scaler/` — Research, proposals & scaling

We are not building "an agent" - we are building the Substrate (The Agentic OS) that allows any world-class agent (Claude, Gemini, Hermes, etc.) to land in this workspace and immediately become 10x more autonomous and capable.

The "Perfect System" is one where the workspace provides the **Senses** (Toolbox Registry), the **Memory** (BOARD.yaml), and the **Muscles** (Toolbox Skills), while the agents provide the "Brain."
