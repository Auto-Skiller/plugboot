# 🏛️ طلب رخصة البلدية — Baladia d'Oran

## الخطوة الأولى والأهم
**لا يمكن البدء بأي عمل بدون الحصول على موافقة البلدية (Baladia) لاستغلال الموقع.**

---

## 📍 معلومات الموقع

| المعلومة | التفاصيل |
|----------|----------|
| **العنوان الدقيق** | برج البحري، الكاب، بالقرب من بلاسيت جاردان دو كاب |
| **الولاية** | وهران (Oran) |
| **البلدية** | بلدية برج البحري (Baladia de Bordj El Bahri) |
| **القطاع** | تجارة الشارع — كشك طعام |
| **المساحة المطلوبة** | 15-25 متر مربع |
| **نوع النشاط** | بيع مأكولات الشارع والمشروبات |

---

## 📋 الوثائق المطلوبة

### 1. الهوية الشخصية
- [ ] بطاقة التعريف الوطنية (CNI) — نسخة + أصل
- [ ] صورة شخصية حديثة (2)
- [ ] شهادة الميلاد (إن طلب)

### 2. السجل التجاري
- [ ] طلب استخراج السجل التجاري (RC)
- [ ] شهادة عدم الإفلاس (إن طلب)
- [ ] مقر النشاط (عنوان الكشك)

### 3. الملف التقني للمشروع
- [ ] وصف النشاط بالتفصيل (قائمة الطعام، ساعات العمل)
- [ ] مخطط الموقع (Plan de situation)
- [ ] صورة للموقع الحالي
- [ ] وصف للمعدات المستخدمة

### 4. الشهادات الصحية
- [ ] شهادة صحية للمالك
- [ ] شهادة صحية للموظفين (بعد التوظيف)
- [ ] شهادة نظافة الموقع (إن طلب)

### 5. التأمينات
- [ ] تأمين المسؤولية المدنية
- [ ] تأمين الحريق (إن طلب)

## 📝 خطوات الإجراء

### المرحلة 1: التحضير (الأسبوع 1-2)
```
1. تجهيز كل الوثائق المذكورة أعلاه
2. حجز موعد مع بلدية برج البحري
3. زيارة الموقع مع مسؤول للتأكد من المطابقة
```

### المرحلة 2: تقديم الطلب (الأسبوع 2-3)
```
1. تقديم ملف طلب الرخصة في البلدية
2. دفع الرسوم الإدارية
3. الحصول على إيصال الوديعة
```

### المرحلة 3: الدراسة والموافقة (الأسبوع 3-6)
```
1. دراسة الملف من طرف مصالح البلدية
2. زيارة ميدانية للموقع (إن طلب)
3. الحصول على قرار الموافقة أو الرفض
```

### المرحلة 4: استلام الرخصة (الأسبوع 6-8)
```
1. استلام شهادة الموافقة النهائية
2. استلام لوحة المحل (إن طلب)
3. بدء النشاط رسمياً
```

1. **المسافة من المؤسسات التعليمية:** لا تقل عن 100 متر من المدارس
