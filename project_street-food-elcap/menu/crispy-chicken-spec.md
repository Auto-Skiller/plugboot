# 🍗 الكريسبي تشيكن الجزائري — Signature Item

## الوصف
دجاج مقرمش متبل بالتوابل الجزائرية الأصيلة، مقلي بتقنية خاصة تعطي قشرة ذهبية مقرمشة مع لحم طري من الداخل. يقدم مع صلصة الثوم والخبز الطازج.

## طريقة التحضير

### 1. التبيل (15 دقيقة)
```
1. قطع الدجاج إلى شرائح متوسطة
2. اخلط التوابل: فلفل أسود، أحمر، كمون، بابريكا، ملح
3. تتبل الدجاج واتركه 10 دقائق
```

### 2. التغليف (5 دقائق)
```
1. اغمس في خليط البيض + الحليب
2. غطس في خليط الدقيق + النشا + التوابل
3. تأكد من التغطية الكاملة
```

### 3. القلي (8-10 دقائق)
```
1. سخن الزيت إلى 170°م
2. اقلي 3-4 دقائق حتى ذهبي
3. اقلب واقلي 3-4 دقائق إضافية
4. صفّي على ورق ماص
```

### 4. التقديم
```
1. صلصة الثوم (زبادي + ثوم + ليمون)
2. خبز طازج
3. سلطة جانبية (اختياري)
```
