# 🧀 الكنافة بالقشطة — Signature Item

## الوصف
كنافة طازجة مقرمشة من الخارج، محشوة بالقشطة الكريمية والجوز المفروم، مغطاة بشراب العسل والورد. حلوى شرقية بلمسة جزائرية أصيلة.

## طريقة التحضير

### 1. تحضير القشطة (5 دقائق)
```
1. اخلط القشطة مع السكر البودرة
2. أضف ماء ورد (ملعقة صغيرة)
3. اتركها تتخثر قليلاً
```

### 2. تحضير الكنافة (10 دقائق)
```
1. قطع عجينة الكنافة إلى خيوط رفيعة
2. اخلطها مع الزبدة المذابة
3. وزع نصف الكمية في الصينية
```

### 3. الحشو والخبز (20 دقيقة)
```
1. رطب الطبقة السفلية بزبدة
2. وزع القشطة بالتساوي
3. رش الجوز المفروم
4. غطي بباقي الكنافة
5. اخبز 180°م لمدة 15-20 دقيقة
```

### 4. التشطيب والتقديم
```
1. اسكب شراب العسل فور الخروج من الفرن
2. زين بالفستق المفروم
3. قدم ساخن (خلال 10 دقائق)
```

## نصائح للجودة

1. **لا تفرط في القلي** — اللون الذهبي المثالي = لا بني غامق
2. **القشطة يجب أن تكون كثيفة** — أضف نشا إذا كانت سائلة
3. **قدم ساخن** — الكنافة تفقد مقرمشتها بعد 15 دقيقة
4. **شراب العسل يكون دافئ** — لا ساخن جداً (يذوب الكنافة)
