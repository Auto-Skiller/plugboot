## What is in this folder?

This folder contains AI role definitions, partitioned by domain. Each file describes an agent persona, behavioral protocols, and capabilities specific to a domain.

## When should the AI read files from here?

Read files from this folder when the task requires delegating to a specialized agent role (e.g., frontend, backend, security).

## How to navigate and use the contents?

- **Structure**: **Domain** -> Semantic routing → prune search space.
- **Format**: `domain/` directories containing `[domain]-[role].md` files.
- Route semantically by role and domain naming to find the specific agent definition needed.
