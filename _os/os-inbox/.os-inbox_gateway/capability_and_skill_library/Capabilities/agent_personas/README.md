# agent_personas

**Pillar:** capability_and_skill_library  **Aspect:** Capabilities

## Role
Reusable expert personas the agent can adopt for a task (architect, SEO, a11y, chief-of-staff, ...).

## Functionalities
- Adopt a persona to specialize tone/expertise
- Pick the right expert for a sub-problem

## when_to_use
When a task maps to the function(s) above.

## triggers
- when a task requires agent personas
- when activating a capability housed in this group
