## What is in this folder?

This folder contains domain skills, capabilities, and workflows, partitioned by domain. Each skill module defines a specific capability that agents can leverage.

## When should the AI read files from here?

Read files from this folder when the task requires a specific capability or domain knowledge workflow that has been explicitly defined as a skill.

## How to navigate and use the contents?

- **Structure**: **Domain** -> Semantic routing by task type → prune search space.
- **Format**: `domain/` directories containing a `skill-name/` module (with `skill.md` and optional helpers).
- Route semantically by domain and task type to find the appropriate skill.
