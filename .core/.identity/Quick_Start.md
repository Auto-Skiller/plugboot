# 🚀 Quick Start

## For Human Operators
1. Set strategic sessions and goals in `CONTROLER.yaml`.
2. Switch to appropriate mode (STRICT/COLLAB/AUTO).
3. Let agents execute autonomously.

## For AI Agents
1. **Read `.brain/meta.router.yaml`** — This is your Master Map. Understand the schemas and operational paths.
2. **Check `CONTROLER.yaml`** — Detect current mode, active sessions, and goals.
3. **Scan `.core/mission_board/` Context** — Read the target files for your assigned goal.
4. **Follow the 10-Step Flow** — Execute tasks deterministically and update `.core/mission_board/` as you go.
