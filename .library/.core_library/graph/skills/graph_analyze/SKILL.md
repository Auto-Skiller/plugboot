# SKILL: graph_analyze

**Engine:** `map_engine` (graphify)
**Runtime:** Python — zero LLM cost
**Trigger:** `/graph-analyze [graph_path]`

## What This Skill Does

Reads an existing `graph.json` and extracts structural insights:
- **God nodes** — most connected abstractions (the core of the codebase)
- **Surprising connections** — non-obvious cross-module links
- **Suggested questions** — what the graph is uniquely positioned to answer
- **Graph diff** — what changed since last build snapshot
- **Knowledge gaps** — isolated nodes, thin communities, ambiguous edges

## When To Use

- After a `graph_build` or `graph_update` to surface insights
- Before a refactoring session — understand what's most connected
- In reports: "what's the architecture of this codebase?"
- Tracking drift: "what changed since last week?"

## Execution

```bash
# Full analysis report (writes to graphify-out/ANALYSIS.md):
python .library/.engines_library/registry_engine/skills/graph_analyze/scripts/analyze.py graphify-out/graph.json

cat graphify-out/GRAPH_REPORT.md
```

## Python API

```python
from graphify.analyze import god_nodes, surprising_connections, suggest_questions, graph_diff
from graphify.cluster import cluster
from graphify.serve import _load_graph, _communities_from_graph

# Load graph
G = _load_graph("graphify-out/graph.json")
communities = _communities_from_graph(G)
community_labels = {cid: f"Community {cid}" for cid in communities}

# God nodes (most connected real entities, file hubs excluded)
gods = god_nodes(G, top_n=10)
# Returns: [{"id": "...", "label": "...", "degree": N}, ...]

# Surprising connections (cross-module, cross-type, cross-community)
surprises = surprising_connections(G, communities, top_n=5)
# Returns: [{"source": "...", "target": "...", "relation": "...", "confidence": "...", "why": "..."}, ...]

# Suggested questions
questions = suggest_questions(G, communities, community_labels, top_n=7)
# Returns: [{"type": "ambiguous_edge|bridge_node|verify_inferred|isolated_nodes|low_cohesion", "question": "...", "why": "..."}, ...]

# Graph diff (compare two snapshots)
import json
G_old = _load_graph("graphify-out/.last-build-snapshot.json")
diff = graph_diff(G_old, G)
# Returns: {"new_nodes": [...], "removed_nodes": [...], "new_edges": [...], "removed_edges": [...], "summary": "..."}
print(diff["summary"])
```

## Diff Detection

```python
# Auto-detect changes > 30% (flag for review before overwriting reports):
old_count = len(G_old.nodes())
new_count = len(G.nodes())
change_pct = abs(new_count - old_count) / max(old_count, 1) * 100
if change_pct > 30:
    print(f"WARNING: Graph changed {change_pct:.1f}% — review before overwriting docs")
```
